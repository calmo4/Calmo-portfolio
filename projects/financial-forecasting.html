Academic Budget Forecasting Model

Simulated budgeting, variance analysis, and year-end forecasting for a mid-size academic unit (~$3–4M).

View GitHub Repo →

Overview

This project simulates financial operations for a university unit across payroll, travel, equipment, and operations. It generates realistic monthly spending behavior and produces budget vs actual reporting, burn-rate based forecasts, and an executive-style narrative summary suitable for leadership.

Key Output

Budget vs. forecast comparison across categories:

Methods

  • Payroll: Normal distribution (stable monthly spend)
  • Travel: Lognormal distribution (right-skew; occasional spikes)
  • Equipment: Gamma distribution (lumpy purchases)
  • Operations: Normal distribution (moderately stable)
  • Budgeting: Annual budgets derived from expected spend + category buffer
  • Forecasting: Burn-rate projection from average monthly spend

Tools

Python, pandas, NumPy (data generation, analysis, forecasting, reporting)
